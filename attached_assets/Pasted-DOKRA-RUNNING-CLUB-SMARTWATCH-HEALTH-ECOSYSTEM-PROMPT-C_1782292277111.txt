DOKRA RUNNING CLUB – SMARTWATCH & HEALTH ECOSYSTEM PROMPT

Continue building the Flutter app DOKRA Running Club.

This app is intended to become a premium Indian fitness platform capable of competing with Nike Run Club and Strava.

Build the smartwatch and health ecosystem with production-level architecture.

PRIMARY GOAL

Users should be able to:

Leave their phone at home.
Track activities from smartwatch.
Automatically sync all activity data into DOKRA.
Receive highly accurate fitness statistics.
Generate share cards and analytics from synced data.

AUTO ACTIVITY DETECTION

If smartwatch records:

Running → create Running activity.

Walking → create Walking activity.

Cycling → create Cycling activity.

No manual action required.

BACKGROUND SYNC

Even when app is closed:

Continue syncing.
Restore missed records.
Sync after reconnecting internet.
DUPLICATE PROTECTION

Prevent duplicate activities.

If same activity exists:

Update existing record instead of creating a new one.
